# 🔐 Password Generator

A simple and secure **Password Generator Web App** built using modern web technologies.
It allows users to generate strong passwords with customizable options.

---

## 🚀 Features

* Generate strong secure passwords
* Choose password length
* Include:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Special characters
* Copy password to clipboard
* Responsive UI

---

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/yourusername/Password_Generator.git
```

Go to project folder

```
cd Password_Generator
```

Install backend dependencies

```
cd backend
npm install
```

Install frontend dependencies

```
cd ../frontend
npm install
```

---

## ▶️ Run the Project

Start backend

```
cd backend
node server.js
```

Start frontend

```
cd frontend
npm run dev
```
